## Как загрузить на GitHub:

1. Скачай файл `index.html`
2. Зайди в репозиторий на GitHub
3. Удали старый `Index.txt`
4. Загрузи новый `index.html` (кнопка **Add file → Upload files**)
5. Settings → Pages → выбери ветку `основной` → Save
6. Сделай репозиторий **публичным** (Settings → Danger Zone → Change visibility)

Через 2-3 минуты сайт будет живой по адресу `https://ТВОЙюзернейм.github.io/tl-takso-bot/`
